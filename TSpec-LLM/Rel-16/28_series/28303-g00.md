Introduction
============

The present document is part of a TS-family covering the 3^rd^
Generation Partnership Project; Technical Specification Group Services
and System Aspects; Telecommunication management; as identified below:

Information Service (IS)**

**28.303 LSA Controller (LC) Integration Reference Point (IRP); Solution
Set (SS) definitions**

1 Scope
=======

The purpose of the present document is to define the mapping of the LSA
Controller (LC) Integration Reference Point (IRP) (see 3GPP TS 28.302
\[2\]) to the protocol specific details necessary for implementation of
this IRP in a CORBA/IDL environment and in a SOAP/WSDL environment.

This Solution Set specification is related to 3GPP TS 28.302 V14.0.X.
